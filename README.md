# IEEE-EMBS-Board-Members
This is the source code for the dashboard for the Board Members of the IEEE-EMBS Chapter at VIT Vellore in our official website 

### Features & Tech Stack
*   Tech Stack: React 18, Tailwind CSS, Framer Motion 11, and Web Audio API.
*   3D Interaction: Cards dynamically tilt in 3D based on mouse movement.
*   Liquid Metal Effect: Animated CSS gradients that simulate real-time metallic reflections.
*   Audio Synthesis: Plays a procedural "card flip" sound effect using native browser audio tools (no external files needed).
*   Responsive Grid: Fully optimized for mobile, tablet, and desktop screens.
